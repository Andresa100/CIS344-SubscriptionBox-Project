## Project Description
This project designs and implements a database for a Subscription Box Service. The system manages customers, subscriptions, boxes, products, payments, and shipments.

The project includes:
- Chen style ER diagram (hand drawn)
- UML ER diagram created in MySQL Workbench
- SQL scripts to create the database and tables
- MySQL Workbench model file
- Final project report

## Repository Structure
- Chen_Diagram/ → Hand-drawn Chen ER diagram
- UML_Diagram/ → UML ER diagram from MySQL Workbench
- SQL_Scripts/ → SQL schema and scripts
- MySQL_Model/ → MySQL Workbench .mwb file
- Report/ → Final project report PDF
